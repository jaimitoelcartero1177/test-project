# AWS Mac Mini Telemetry Tool v2.1.0
High-performance monitoring tool.

## INSTALLATION NOTICE (Mac Mini)
Due to a change in the log architecture, version v2.1.0 is incompatible with v1.x session data that was stored in the `Documents` folder.

To avoid critical deployment failures, **it is mandatory to clean the legacy logs directory** described in **Issue #1** before proceeding.
